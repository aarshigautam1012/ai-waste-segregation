# ♻️ AI Waste Segregation MVP

## Problem Statement
Improper waste segregation causes environmental pollution and inefficient recycling.
People often do not know which bin to use for different waste items.

## Solution
This project is a Minimum Viable Product (MVP) that helps users identify waste type
and suggests the correct disposal method using a simple web application.

## Features
- Identifies waste category
- Suggests correct bin
- Easy to use interface
- Free and lightweight
- Hackathon-ready MVP

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Waste Classification Examples
- Banana Peel → Organic Waste → Green Bin
- Plastic Bottle → Dry Waste → Blue Bin
- Metal Can → Recyclable → Recycling Bin
- Battery → E-Waste → Authorized E-Waste Center
- Glass Bottle → Dry Waste → Blue Bin

## How to Run the Project
1. Download or clone the repository
2. Open terminal in project folder
3. Run `npm install`
4. Run `node server.js`
5. Open browser and go to `http://localhost:3000`
